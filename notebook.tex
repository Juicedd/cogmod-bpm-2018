    The following function calculates makes a decision for one day at the
factory. For the first 10 days it just picks a random number of workers
in order to create some examples in memory. After that, it will use
partial matching to retrieve an experience form memory that is most
similar to the current situation. That is: it retrieves an experience
with the current production as "yesterday", today's production at 9
(because that is the goal). It then uses the retrieved number of workers
as its decision. The result is then added to declarative memory.
